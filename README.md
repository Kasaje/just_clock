# 🕰️ Just Clock: Glass Edition

A high-end, luxury timepiece web application inspired by the **Apple Watch Ultra** and **Glassmorphism** design principles. Built with precision, fluidity, and responsiveness in mind.

[**Live Demo »**](https://just-clock-sand.vercel.app/)

---

## ✨ Features

- **Glassmorphism UI:** Multi-layered backdrop blurs and specular highlights to simulate a premium Sapphire Crystal watch face.
- **Mechanical Sweep Motion:** A buttery-smooth second hand that mimics high-end mechanical movements using `requestAnimationFrame`.
- **Apple-Inspired Complications:** Minimalist date and time complications featuring the signature "International Orange" accents.
- **Fully Responsive:** Perfectly scaled for mobile, tablet, and desktop displays using Tailwind CSS.
- **Performance First:** Optimized with Next.js and React hooks for zero lag and high frame rates.

---

## 🚀 Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS v4
- **Fonts:** Geist Sans
- **Deployment:** Vercel
